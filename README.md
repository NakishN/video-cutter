# 🎬 Нарезчик видео

Автоматическая транскрипция Twitch-VOD и YouTube-видео с выделением ключевых моментов с помощью ИИ.

**Как работает:**
1. **Загрузка:** Загружаете видео/аудио файл с ПК или вставляете ссылку на Twitch / YouTube (приложение скачает только аудиодорожку, что экономит трафик и ускоряет процесс **в 10–50 раз**).
2. **Транскрипция:** [Whisper](https://github.com/ggerganov/whisper.cpp) транскрибирует речь локально (на видеокарте GPU с CUDA или на процессоре CPU с автоматическим резервированием).
3. **Анализ:** [GenAPI](https://gen-api.ru) (GPT-4.1 или Gemini) находит самые интересные, эмоциональные моменты и выдаёт список с таймкодами и описанием.
4. **Экспорт:** Скачиваете готовый архив с текстом реплик, таймкодами (SRT) и резюме.

---

## ✨ Новые возможности и улучшения

* **🎬 Автоматическая нарезка клипов ИИ:** Система анализирует видео с помощью ИИ (через GenAPI) и сама вырезает наиболее интересные, смешные и вирусные моменты в отдельные видеофайлы.
* **✂️ Ручная нарезка в один клик:** Прямо в интерфейсе можно указать точные таймкоды начала/конца и название, чтобы мгновенно вырезать нужный фрагмент из исходного видео.
* **📱 Интеллектуальный кроп под вертикальный формат (9:16):**
  * **Рилсы/Шортсы:** Умное отслеживание лиц (Face-Tracking) через OpenCV для автоматического удержания говорящего в центре вертикального кадра.
  * **Gameplay + Facecam:** Автоматическое разделение экрана (верхняя часть — геймплей по центру, нижняя — веб-камера стримера).
  * **Vtuber:** Вертикальное кадрирование со смещением для аватаров (15% слева).
* **💎 Премиальный дизайн:** Полностью обновлённый интерфейс в тёмных тонах с эффектом Glassmorphism (матовое стекло), интерактивными графиками и плавной анимацией.
* **📺 Полноценная поддержка YouTube и Twitch:** Вставляйте ссылки на видео/стримы, программа скачает только аудиодорожку для ускорения обработки в 10–50 раз.
* **⚡ Аппаратное ускорение (CUDA / NVENC):** Поддержка видеокарт NVIDIA для мгновенной транскрипции через Whisper и быстрого рендеринга видео клипов.
* **📂 Очистка кэша:** Удобная кнопка в меню для удаления временных файлов, чтобы они не занимали место на жестком диске.

## 🪟 Windows — Подробная инструкция (для чайников)

Эта инструкция написана специально для тех, кто никогда не работал с консолью или программированием. Мы разберём 3 простых способа запуска программы.

### 🚀 Способ 1: Автоматический запуск (Самый простой)

Этот способ автоматически настроит программу за 1 клик.

#### **Шаг 1. Установите Python**
1. Перейдите на официальный сайт: [python.org/downloads](https://www.python.org/downloads/)
2. Нажмите большую жёлтую кнопку **Download Python 3.x.x**.
3. Запустите скачанный установщик.
4. > [!IMPORTANT]
   > В самом первом окне установщика внизу **обязательно поставьте галочку "Add python.exe to PATH"** (Добавить в PATH). Без этой галочки Windows не сможет запустить программу.
5. Нажмите **Install Now** и дождитесь завершения.

#### **Шаг 2. Скачайте и распакуйте программу**
1. Нажмите зелёную кнопку **Code** вверху этой страницы, а затем нажмите **Download ZIP**.
2. Распакуйте скачанный ZIP-архив в любую папку на вашем компьютере. 
   > [!TIP]
   > Желательно распаковывать программу по пути, который не содержит русских букв (например, `C:\VideoCutter` или `D:\Soft\VideoCutter` вместо `C:\Пользователи\Иван\Рабочий стол\Нарезчик`). Это предотвратит возможные проблемы с путями в библиотеках.

#### **Шаг 3. Запустите программу**
1. Зайдите в распакованную папку программы и запустите двойным кликом файл **`Запустить.bat`**.
   > *Если Windows покажет предупреждение ("Система Windows защитила ваш ПК"), нажмите ссылку **Подробнее**, а затем кнопку **Выполнить в любом случае**.*
2. При первом запуске откроется черное окно консоли. Скрипт сам создаст виртуальное окружение и установит нужные пакеты. Это займёт 1–3 минуты.
3. По завершении установки скрипт создаст файл `.env` и автоматически откроет его в Блокноте, а в консоли попросит ввести API-ключ.
4. Зарегистрируйтесь на сайте [gen-api.ru](https://gen-api.ru), перейдите в раздел ключей, скопируйте ваш API-ключ (он начинается на `sk-`) и вставьте его в Блокнот после знака `=`:
   `GEN_API_KEY=sk-ваш_ключ_от_gen_api`
5. Сохраните файл (`Ctrl + S`) и закройте Блокнот.
6. Нажмите любую клавишу в окне консоли. Программа запустится и автоматически откроет веб-интерфейс в вашем браузере по адресу `http://127.0.0.1:8000`.

### 🛠️ Способ 2: Полностью ручная установка (Если авто-запуск выдаёт ошибки)

Если PowerShell или антивирус блокируют автоматический скрипт, вы можете скачать и установить всё вручную.

#### **Шаг 1. Установите Python**
Установите Python, как описано в Шаге 1 Автоматического запуска. **Не забудьте галочку "Add python.exe to PATH"!**

#### **Шаг 2. Скачайте программу**
Скачайте и распакуйте ZIP-архив с кодом программы в удобную папку (например, `C:\VideoCutter`).

#### **Шаг 3. Скачайте FFmpeg (для нарезки видео)**
1. Перейдите по ссылке: [FFmpeg Builds (GitHub)](https://github.com/BtbN/FFmpeg-Builds/releases/latest)
2. Скачайте архив **`ffmpeg-master-latest-win64-gpl.zip`**.
3. Откройте его, перейдите в папку `ffmpeg-master-latest-win64-gpl`, затем в `bin`.
4. Скопируйте файлы **`ffmpeg.exe`** и **`ffprobe.exe`** и вставьте их **прямо в основную папку программы** (рядом с файлом `server.py`).

#### **Шаг 4. Скачайте Whisper.exe и модель**
1. Перейдите по ссылке: [Whisper.cpp Releases](https://github.com/ggml-org/whisper.cpp/releases/tag/v1.7.5)
2. Скачайте архив **`whisper-blas-bin-x64.zip`**.
3. Распакуйте его и скопируйте файл **`main.exe`** в папку с нашей программой, переименовав его в **`whisper.exe`**.
4. В папке программы создайте папку с именем **`models`**.
5. Скачайте файл модели по ссылке: [Скачать ggml-medium.bin (1.5 ГБ)](https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-medium.bin) и положите его внутрь папки `models`.

#### **Шаг 5. Создайте файл ключа (.env)**
1. Скопируйте файл `.env.example` и назовите копию `.env`.
2. Откройте `.env` через Блокнот и запишите ваш API-ключ: `GEN_API_KEY=sk-ваш_ключ`. Сохраните и закройте.

#### **Шаг 6. Установите библиотеки и запустите**
1. В проводнике Windows откройте папку с программой.
2. В адресной строке вверху сотрите текущий путь, напишите **`cmd`** и нажмите **Enter**. Откроется консоль.
3. Скопируйте и вставьте в консоль по очереди следующие команды:
   ```cmd
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   python launcher.py
   ```
4. Откройте в браузере `http://127.0.0.1:8000`.

### 📦 Способ 3: Сборка в один автономный `.exe` файл

Если вы хотите запускать программу без необходимости устанавливать Python на других компьютерах, вы можете собрать её в полноценное Windows-приложение.

1. Запустите файл **`setup_windows.bat`** (он подготовит все зависимости, скачает ffmpeg и whisper).
2. Запустите файл **`build_windows.bat`**. Начнётся процесс компиляции (это займет 1-3 минуты).
3. По завершении в корневой папке появится каталог **`dist\НарезчикВидео`**.
4. Запустите файл **`НарезчикВидео.exe`** из этой папки. Всё готово! Вы можете перенести папку `НарезчикВидео` на любой другой компьютер с Windows, и программа будет работать автономно.

### 🔍 Решение частых проблем на Windows

| Проблема | Причина | Решение |
|---|---|---|
| **Окно консоли открывается и сразу закрывается** | Python не установлен или не поставлена галочка PATH | Переустановите Python. Запустите установщик, нажмите **Modify** (или удалите и установите заново) и убедитесь, что внизу отмечен пункт **Add python.exe to PATH**. |
| **Ошибка "ModuleNotFoundError: No module named 'cv2'"** | Не установились библиотеки | Откройте консоль в папке проекта и выполните вручную: `venv\Scripts\pip install -r requirements.txt` |
| **Не нарезаются видео (ошибка FFmpeg)** | Не найдены файлы ffmpeg.exe или ffprobe.exe | Убедитесь, что файлы `ffmpeg.exe` и `ffprobe.exe` лежат прямо в корневой папке проекта (рядом с `server.py`). Если вы собирали `.exe` через PyInstaller, убедитесь, что они скопированы в папку `dist/НарезчикВидео/`. |
| **Ключ не принимается / Ошибка авторизации** | Ключ вписан неверно или не сохранен файл `.env` | Откройте файл `.env` в Блокноте. Проверьте, что строка выглядит строго как `GEN_API_KEY=sk-вашкод` (без кавычек, пробелов и скобок). Сохраните файл через Ctrl + S. |
| **Система Windows заблокировала файл** | Ложное срабатывание встроенного защитника SmartScreen на новые скрипты | Нажмите кнопку **"Подробнее"** в окне предупреждения Windows, а затем кнопку **"Выполнить в любом случае"**. Это безопасно. |

## 🐧 Linux — локальный запуск

### Требования

- Ubuntu 20.04+ / Debian 11+ / любой Linux
- Python 3.9+
- ffmpeg
- CUDA (опционально, для ускорения Whisper)

### Шаг 1 — Системные зависимости

```bash
sudo apt-get update
sudo apt-get install -y ffmpeg zip wget git cmake make g++ libopenblas-dev
```

### Шаг 2 — Скачайте проект

```bash
git clone https://github.com/NakishN/video-cutter.git
cd video-cutter
```

### Шаг 3 — Python-окружение

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Шаг 4 — Скомпилируйте Whisper

```bash
cd whisper.cpp
cmake -B build -DGGML_BLAS=ON -DGGML_BLAS_VENDOR=OpenBLAS
cmake --build build --config Release -j$(nproc)
cp build/bin/whisper-cli ../whisper
cd ..
```

> Если есть NVIDIA GPU:
> ```bash
> cmake -B build -DGGML_CUDA=ON
> cmake --build build --config Release -j$(nproc)
> cp build/bin/whisper-cli ../whisper-cuda
> ```
> И поставьте `"use_gpu": true` в `config.json`.

### Шаг 5 — Скачайте модель Whisper

```bash
mkdir -p models
wget https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-medium.bin \
     -O models/ggml-medium.bin
```

Альтернативные модели (меньше = быстрее, но хуже качество):

| Модель | Размер | Качество | Скорость (CPU) |
|---|---|---|---|
| `ggml-tiny.bin` | 75 МБ | низкое | ~2 мин/час видео |
| `ggml-base.bin` | 142 МБ | среднее | ~5 мин/час видео |
| `ggml-small.bin` | 466 МБ | хорошее | ~10 мин/час видео |
| `ggml-medium.bin` | 1.5 ГБ | **отличное** | ~20 мин/час видео |

### Шаг 6 — Настройте .env

```bash
cp .env.example .env
nano .env
```

Вставьте ключ:
```
GEN_API_KEY=sk-ваш-ключ
```

### Шаг 7 — Запуск

```bash
bash start.sh
```

Откройте в браузере: `http://localhost:8000`

## 🌐 Linux — деплой на VPS (Docker)

Подходит если хотите использовать сервис круглосуточно из браузера с любого устройства.

### Требования к серверу

| | Минимум | Рекомендуется |
|---|---|---|
| CPU | 2 ядра | 4 ядра |
| RAM | 4 ГБ | 8 ГБ |
| Диск | 10 ГБ | 20 ГБ |
| ОС | Ubuntu 22.04 | Ubuntu 22.04 |

**Стоимость:** ~1 000–1 500 ₽/мес (Selectel, Timeweb, Hetzner).

### Деплой одной командой

Скопируйте проект на сервер и выполните:

```bash
# С вашей локальной машины — копируем проект на сервер
scp -r /путь/к/video-cutter user@IP_СЕРВЕРА:/opt/video-cutter

# Подключаемся к серверу
ssh user@IP_СЕРВЕРА

# Запускаем деплой
cd /opt/video-cutter
GEN_API_KEY=sk-ваш-ключ sudo -E bash deploy.sh
```

Скрипт сам:
- Установит Docker
- Скачает модель Whisper (~1.5 ГБ)
- Соберёт Docker-образ (5–10 мин при первом запуске)
- Запустит контейнер
- Проверит работоспособность

После запуска приложение доступно по адресу: `http://IP_СЕРВЕРА:8000`

### Управление контейнером

```bash
# Посмотреть статус
docker compose ps

# Посмотреть логи
docker compose logs -f

# Перезапустить
docker compose restart

# Остановить
docker compose down

# Обновить после изменений в коде
docker compose up -d --build
```

### Возможные проблемы (Linux / Docker)

| Проблема | Решение |
|---|---|
| `GEN_API_KEY не задан` | Отредактируйте `/opt/video-cutter/.env`, добавьте ключ, перезапустите |
| Нет места на диске | Воспользуйтесь кнопкой «Очистить кэш видео» в веб-интерфейсе или выполните: `rm /opt/video-cutter/videos/*` |
| Контейнер падает | Проверьте логи: `docker compose logs --tail=50` |
| Порт 8000 занят | Измените в `docker-compose.yml`: `"8080:8000"` |

## ⚙️ Настройки (config.json)

```json
{
  "whisper_model_path": "models/ggml-medium.bin",
  "genapi_network_id": "gpt-4-1",
  "models_dir": "models",
  "use_gpu": false,
  "whisper_language": "ru",
  "video_dir": "videos",
  "output_dir": "output",
  "temp_dir": "tmp"
}
```

| Параметр | Описание |
|---|---|
| `whisper_model_path` | Путь к модели Whisper |
| `genapi_network_id` | Модель GenAPI: `gpt-4-1` или `gemini-2-5-flash-lite` |
| `use_gpu` | `true` — Whisper на GPU (нужна NVIDIA + CUDA), `false` — CPU |
| `whisper_language` | Язык транскрипции: `ru`, `en`, `auto` |
